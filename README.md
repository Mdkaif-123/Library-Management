# Library Management System

Welcome to the Library Management System project! This Python application is designed to help manage a library's book inventory, allowing users to add, search, borrow, and return books.

## Installation

1. Clone the repository or download the project files.
2. Make sure you have Python 3 installed on your system.
3. Install the required dependencies by running the following command:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Open a terminal or command prompt and navigate to the project directory.
2. Run the following command to start the application:
   ```
   python main.py
   ```
3. The application will display a menu with various options for managing the library's books. Follow the on-screen instructions to perform different operations.

## Features

- Add new books to the library's inventory.
- Search for books by title, author, or ISBN.
- Display the available books in the library.
- Borrow books by entering the book's ISBN.
- Return borrowed books by entering the book's ISBN.
- View the list of borrowed books and the borrower's information.
